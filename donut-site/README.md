# Spinning Donut

A minimal Three.js website featuring a 3D donut (torus + icing + sprinkles) that continuously tumbles through all axes.

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

Import this directory as a project on [Vercel](https://vercel.com/new). It's a static Vite app — Vercel auto-detects the framework, or uses the settings in `vercel.json` (`npm run build`, output in `dist/`). No environment variables are required.
